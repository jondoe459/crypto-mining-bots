# Crypto Mining Bots

## Usage
To use the crypto mining bots, simply follow the instructions below:
1. Clone the repository:
   ```bash
   git clone https://github.com/jondoe459/crypto-mining-bots.git
   ```
2. Navigate to the project directory:
   ```bash
   cd crypto-mining-bots
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the mining bots:
   ```bash
   npm start
   ```

## Configuration
Configuration options can be found in the `config.json` file. Here are some key options:
- `miningPool`: The URL of the mining pool.
- `walletAddress`: Your wallet address for receiving earnings.
- `threads`: Number of threads to run.

Make sure to update your `config.json` with your own settings before starting the bots.
